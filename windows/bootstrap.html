Instalando o Windows por "boostrap"

Introdução

Se você lida com manutenção de computadores e tem que instalar o Windows de tempos em tempos, com a mais absoluta certeza você já perdeu uma instalação e teve que reiniciar por algum erro bizarro que o Windows teve, tudo se resolveria se você pudesse usar as ferramentas de uma máquina que já está com o Windows para passar a imagem de sistema Windows direto para um novo disco sem passar pelo processo de instalação... E é o que vamos fazer aqui!

O que você vai precisar:

  • Um computador que já tenha o Windows instalado;

  • Conectar o disco em que você queira fazer a nova instalação neste computador que já tem o Windows instalado;

  • Ter baixado uma ISO da versão do Windows que você quer instalar no novo disco;

Instalação em MBR/BIOS

Preparar disco

Obviamente precisamos limpar o disco alvo e fazer nele a tabela de partição em MBR, eu vou presumir que você conhece o diskpart e não vou explicar comando a comando. Obviamente substitua o X pelo número do disco desejado.

diskpart
list disk
select disk X
clean
convert mbr

Partição de Sistema

Faremos aqui a partição ativa que guarda o código de inicialização do sistema e atribuir a ela a letra S.

create partition primary size=100
format quick fs=ntfs label="System"
assign letter=S
active

Partição do Windows

E agora vamos criar a partição do próprio Windows reservando 600 MB ao final do disco para deixar a partição de Recovery, atribuindo a ela a letra W.

create partition primary
shrink minimum=600
format quick fs=ntfs label="Windows"
assign letter=W

Partição de Recovery

E agora vamos criar a partição oculta de Recovery atribuindo a ela a letra R.

create partition primary
format quick fs=ntfs label="WinRE"
assign letter=R
set id=27

Com as partições prontas quite o diskpart.

exit

Aplicar imagem de sistema

Vamos supor que você montou a ISO de instalação no E:, essa ISO geralmente guarda diversas versões do Windows, liste as imagens contidas na ISO com o comando abaixo:

dism /Get-WimInfo /WimFile:E:\sources\install.wim

Vamos supor que você queira instalar o Windows Home Edition que é o item 1 da lista, para aplicar essa imagem específica para a partição W que preparamos basta usar o comando abaixo:

dism /Apply-Image /ImageFile:E:\sources\install.wim /Index:1 /ApplyDir:W:\

Em algumas isos o nome do arquivo muda, nesse caso troque install.wim por install.esd se necessário.

Configurar inicialização

Preparar a MBR:

E:\boot\bootsect.exe /nt60 S: /mbr

Preparar o bootloader:

bcdboot W:\Windows /s S: /f BIOS

Partição de Recovery

mkdir R:\Recovery\WindowsRE copy W:\Windows\System32\Recovery\Winre.wim R:\Recovery\WindowsRE\Winre.wim

Este último passo só pode ser feito pelo próprio sistema instalado após o primeiro boot!

reagentc /setreimage /path R:\Recovery\WindowsRE /target W:\Windows

Instalação em GPT/UEFI

Preparar disco

Obviamente precisamos limpar o disco alvo e fazer nele a tabela de partição em GPT, eu vou presumir que você conhece o diskpart e não vou explicar comando a comando. Obviamente substitua o X pelo número do disco desejado.

diskpart
list disk
select disk X
clean
convert gpt

Partição ESP

Faremos aqui a partição que guarda o .efi de inicialização do sistema e atribuir a ela a letra S.

create partition efi size=300
format quick fs=fat32 label="System"
assign letter=S

Partição MSR

Para fazer a partição MSR use o comando abaixo.

create partition msr size=16

Partição do Windows

E agora vamos criar a partição do próprio Windows reservando 600 MB ao final do disco para deixar a partição de Recovery, atribuindo a ela a letra W.

create partition primary
shrink minimum=600
format quick fs=ntfs label="Windows"
assign letter=W

Partição de Recovery

E agora vamos criar a partição oculta de Recovery atribuindo a ela a letra R.

create partition primary
format quick fs=ntfs label="WinRE"
assign letter=R
set id="de94bba4-06d1-4d40-a16a-bfd50179d6ac"
gpt attributes=0x8000000000000001

Com as partições prontas quite o diskpart.

exit

Aplicar imagem de sistema

Vamos supor que você montou a ISO de instalação no E:, essa ISO geralmente guarda diversas versões do Windows, liste as imagens contidas na ISO com o comando abaixo:

dism /Get-WimInfo /WimFile:E:\sources\install.wim

Vamos supor que você queira instalar o Windows Home Edition que é o item 1 da lista, para aplicar essa imagem específica para a partição W que preparamos basta usar o comando abaixo:

dism /Apply-Image /ImageFile:E:\sources\install.wim /Index:1 /ApplyDir:W:\

Em algumas isos o nome do arquivo muda, nesse caso troque install.wim por install.esd se necessário.

Configurar inicialização

Preparar o bootloader:

bcdboot W:\Windows /s S: /f UEFI

Partição de Recovery

mkdir R:\Recovery\WindowsRE copy W:\Windows\System32\Recovery\Winre.wim R:\Recovery\WindowsRE\Winre.wim
